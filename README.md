# Web‑Stud — Modern website for a web studio

A responsive promo site created to showcase the services of a web studio.
The project was designed with an emphasis on clean design, logical structure, and smooth animations.

## Features

- Responsive design
- Burger menu
- Smooth section transitions
- Service presentation blocks
- Modern layout with clean UI
- Optimized images
- Contact form

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, animations)
- JavaScript
- Responsive Web Design
- GitHub Pages (deployment)
- Netlify (secondary deployment)

## My Role

- UI/UX design — creating the structure, logic, and visual style of the website without using Figma, fully based on personal design vision.

- Layout development — building all sections with clean, well‑organized HTML and CSS.

- Responsive design — adapting the website for mobile devices, tablets, and large screens.

- Animations — adding smooth transitions between sections and subtle interaction effects.

- Image optimization — reducing image weight for faster loading and better performance.

- Deployment preparation — preparing project files and structure for publishing on GitHub Pages and Netlify.
